# Blind Auction 🏷️

## Overview

This project is a **Blind Auction program written in Python** that allows multiple users to place secret bids. Each bidder enters their name and bid amount, and the program determines the highest bidder at the end of the auction.

A blind auction is a type of auction where bidders submit bids without knowing the bids of others, and the highest bid wins.

---

## Features

* Collects bidder names and bid amounts
* Stores bids using a Python dictionary
* Allows multiple participants to place bids
* Clears the screen between bids to keep bids private
* Automatically determines the highest bidder

---

## Technologies Used

  * Functions
